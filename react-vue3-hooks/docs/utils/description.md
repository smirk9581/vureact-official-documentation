# 工具集定位与使用指南

本工具集侧重于作为 **VuReact 编译器**的**运行时核心（Runtime Core）**。

其主要职责是在编译阶段，为 Vue 的指令提供必要的底层 API 转换支持，确保最终生成的 React JSX 代码能够高效、稳定地在运行时环境中工作。

## 关于直接使用

1. **通用工具（推荐）：**
    * **`nextTick`** 以及所有 **无前缀** 的 Hook（如 `useWatch`、`useState$`、`useMounted` 等）被设计为通用的、语义清晰的开发工具。它们可以**直接导入**并在任何标准 React 项目中使用，享受 Vue 风格的开发体验。

2. **低级 API（编译器专用）：**
    * 大多数以 `v` 开头的工具方法，属于为 `VuReact` 编译器内部流程设计的**低级 API（Low-Level Primitives）**。
    * 这些 API 具备较高的抽象性和专业性，如果选择在业务代码中直接调用，你需要对 `VuReact` 的编译转换机制有深入的了解和适应过程。我们建议普通开发者**避免直接使用**这些工具，除非你正在扩展或调试 `VuReact` 编译器本身。
