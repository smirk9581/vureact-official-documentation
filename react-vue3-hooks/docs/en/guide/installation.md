# Installation

## 1. Environment Requirements

This library depends on the following versions of the environment. Make sure your project meets the following conditions before installation:

- React: `18.2.0` or higher
- Node.js: `20.0.0` or higher

## 2. Installation Methods

Three mainstream package management tools are supported: npm, pnpm, and yarn. You can choose the corresponding installation command according to your project habits:

### npm

```bash
npm install react-vue3-hooks
```

### pnpm

```bash
pnpm add react-vue3-hooks
```

### yarn

```bash
yarn add react-vue3-hooks
```
