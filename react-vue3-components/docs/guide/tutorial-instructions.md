# 本教程的约定

- 文档中的所有示例代码均只展示重要代码或者伪代码，以此来突出重点。

- 展示的 Demo 示例均存放在 [CodeSandBox](https://codesandbox.io/p/sandbox/intelligent-dawn-w4c2h5) 网站，里面包含了所有完整的代码实现，亦或者可以通过 Demo 的 `Open SandBox` 按钮查看对应源代码。

- 所有示例与代码均与 [Vue3 内置组件](http://vcn.vuejs.org/guide/built-ins/transition.html)的官方文档和 [Vue Router](https://router.vuejs.org/zh/introduction.html) 的官方文档对齐，达到几乎一致。

- 我们假设你已经对 **React** 、**Vue** 、**React Router DOM** 和 **Vue Router** 有了一定的了解。
