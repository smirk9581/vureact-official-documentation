# Installation

## 1. Environment Requirements

This library depends on the following versions of the environment. Make sure your project meets the following conditions before installation:

- React: `18.0.0` or higher
- Node.js: `20.0.0` or higher

## 2. Installation Methods

Three mainstream package management tools (npm, pnpm, yarn) are supported. You can choose the corresponding installation command according to your project habits:

### npm

```bash
npm install react-vue3-components
```

### pnpm

```bash
pnpm add react-vue3-components
```

### yarn

```bash
yarn add react-vue3-components
```
